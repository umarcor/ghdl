pyGHDL.dom.Aggregates¶

This module contains all DOM classes for VHDL’s design units (context, architecture, package, package body, context and configuration.

Classes

  • SimpleAggregateElement: A AggregateElement is a base-class for all aggregate elements.

  • IndexedAggregateElement: A AggregateElement is a base-class for all aggregate elements.

  • RangedAggregateElement: A AggregateElement is a base-class for all aggregate elements.

  • NamedAggregateElement: A AggregateElement is a base-class for all aggregate elements.

  • OthersAggregateElement: A AggregateElement is a base-class for all aggregate elements.
